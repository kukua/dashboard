# Kukua Dashboard

## Usage

```bash
git clone https://github.com/kukua/dashboard.git
cd dashboard/
cp src/js/config.js.example src/js/config.js
chmod 600 src/js/config.js

sudo npm install -g grunt-cli
npm install
grunt copy
grunt build
```
